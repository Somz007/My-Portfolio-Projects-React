# Project 02 — GitHub Jobs App

A React application that lets developers search and filter job listings using the GitHub Jobs API (or a compatible proxy).

## Tech Stack

- **Framework:** React 18
- **Build Tool:** Vite
- **Styling:** CSS Modules
- **API:** GitHub Jobs API / Remotive API
- **HTTP Client:** Axios

## Features

- Search jobs by keyword, location, and full-time filter
- Paginated results
- Job detail view with markdown-rendered description
- Responsive card-based layout

## Getting Started

```bash
cd project-02-github-jobs-app
npm install
npm run dev
```
