# Project 03 — Photo Gallery

A React photo gallery application with masonry layout, lightbox view, and optional Unsplash API integration.

## Tech Stack

- **Framework:** React 18
- **Build Tool:** Vite
- **Styling:** CSS Modules
- **API:** Unsplash API (optional)

## Features

- Masonry/grid photo layout
- Lightbox modal on image click
- Search photos by keyword
- Lazy loading for performance
- Responsive design

## Getting Started

```bash
cd project-03-photo-gallery
npm install
npm run dev
```

Create a `.env` file at the project root:

```
VITE_UNSPLASH_ACCESS_KEY=your_access_key_here
```

Get a free API key at [https://unsplash.com/developers](https://unsplash.com/developers).
