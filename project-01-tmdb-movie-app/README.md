# Project 01 — TMDB Movie App

A React application that fetches and displays movies, TV shows, and cast details using the TMDB (The Movie Database) API.

## Tech Stack

- **Framework:** React 18
- **Build Tool:** Vite
- **Styling:** CSS Modules
- **API:** TMDB REST API (v3)
- **HTTP Client:** Axios

## Features

- Browse trending, top-rated, and upcoming movies
- Search movies and TV shows
- View detailed info: overview, ratings, cast, trailers
- Responsive grid layout

## Getting Started

```bash
cd project-01-tmdb-movie-app
npm install
npm run dev
```

Create a `.env` file at the project root:

```
VITE_TMDB_API_KEY=your_api_key_here
```

Get a free API key at [https://www.themoviedb.org/settings/api](https://www.themoviedb.org/settings/api).
