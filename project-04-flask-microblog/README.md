# Project 04 — Flask Microblog

A lightweight microblogging web application built with Python and Flask, featuring user posts, a simple feed, and SQLite persistence.

## Tech Stack

- **Backend:** Python 3 / Flask
- **Database:** SQLite (via Flask-SQLAlchemy)
- **Templating:** Jinja2
- **Styling:** Vanilla CSS
- **Auth:** Flask-Login

## Features

- User registration and login
- Create, read, and delete posts
- Personal post feed
- Flash messaging for user feedback
- Responsive layout

## Getting Started

```bash
cd project-04-flask-microblog
python -m venv venv
# Windows
venv\Scripts\activate
# macOS / Linux
source venv/bin/activate

pip install -r requirements.txt
python app.py
```
